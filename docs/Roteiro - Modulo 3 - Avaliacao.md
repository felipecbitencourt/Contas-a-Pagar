**Slide 38: 2. No macrofluxo do contas a Pagar, na compra de mercadorias, qual é a sequência correta de etapas?**

A) Entrega → Solicitação → Lançamento → Pagamento → Conciliação

**Slide 39: 3. Sobre as formas de pagamento, qual alternativa está correta?**
